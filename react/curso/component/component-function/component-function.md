# index.jsx

# webpack.config.js
resolve -> extensions para fazer import sem precisar colocar a extensão do aquivo.

```
resolve: {
    extensions: ['', '.js', '.jsx'] /* para não precisar ficar colocando a extensão no import */
}
```
